# Todo List 视觉设计方案

**日期:** 2026-06-11
**状态:** 已确认

## 1. 设计方向

**温和舒展** — 保留工具型产品的清晰结构，通过更大的阅读字号、克制的柔和蓝、轻量字段行和响应式分栏降低长时间使用的视觉疲劳。

## 2. 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主题色 | `#4F6FAE` | 选中态、按钮、链接、checkbox |
| 主题浅色 | `#EDF2F8` | 选中行背景、hover 态 |
| 主题深色 | `#405F9C` | 按钮 hover、按压态 |
| 页面背景 | `#F4F6F8` | 全局底色 |
| 卡片/面板背景 | `#FFFFFF` | 中栏、右栏 |
| 侧栏背景 | `#F5F7FA` | 左栏浅灰蓝底 |
| 边框色 | `#DDE3E9` | 列分隔、输入框边框 |
| 主文字 | `#29313D` | 标题、任务正文 |
| 次文字 | `#525D6B` | 菜单项、字段值 |
| 辅助文字 | `#7B8592` | 计数、字段标签 |
| 禁用/占位 | `#9BA4AF` | 搜索占位、禁用态 |
| 高优先级 | `#C96F43` | 高优先级 checkbox |
| 中优先级 | `#C08A32` | 中优先级 checkbox |
| 低优先级 | `#5F7FB6` | 低优先级 checkbox |
| 完成/删除 | `#7B8592` + 删除线 | 已完成任务 |
| 标签背景 | `#EEF1F4` | 标签 pill |

## 3. 字体

- 主字体: `Segoe UI, system-ui, -apple-system, sans-serif`
- 代码/等宽: `Cascadia Code, Consolas, monospace`
- 字号层级: 12/13/14/16/18/22px
- 任务、输入框和字段值以 `14px` 为主；辅助信息不低于 `12px`
- 行高: 1.45（任务列表），1.6（描述正文）

## 4. 圆角与阴影

| 元素 | 圆角 |
|------|------|
| 卡片/面板 | 12px |
| 按钮/输入框 | 8–10px |
| 选中行背景 | 9px |
| 标签 pill | 14px（全圆角） |
| checkbox | 5px |
| 小徽章 | 3px |

阴影仅用于弹层、临时侧栏和详情抽屉，常驻面板依靠背景及细边框分层。

## 5. 布局参数

- 左栏展开宽度: `240px`，收起宽度: `64px`
- 中栏最小宽度: `420px`
- 右栏宽度: 宽屏下随窗口在 `420–800px` 间伸缩并始终显示；无选中任务时显示空状态
- 中栏标题、快速添加和任务列表铺满中栏可用宽度，保留两侧内容间距
- 任务行最小高度: `50px`
- 区块间距: `14-18px`

### 5.1 响应式
- `>= 1280px`: 完整三栏，右侧详情栏常驻，尊重用户保存的侧栏展开状态。
- `1120–1279px`: 左栏自动收为 `64px` 图标栏，右侧详情栏常驻；点击左栏后临时覆盖展开，不修改保存偏好。
- `< 1120px`: 左栏保持图标栏，任务详情改为带遮罩的右侧抽屉。
- 抽屉支持关闭按钮、点击遮罩和 `Escape` 关闭。

## 6. 组件样式

### 6.1 Checkbox
- 未选中: `18×18px`，`2px solid #B9C2CC` 边框，`5px` 圆角
- 无优先级使用主题蓝；低/中/高优先级分别使用蓝、琥珀、橙色
- 已选中填充当前优先级色并显示白色对勾

### 6.2 按钮
- 主按钮: 填充 `#4F6FAE`，白色文字，`9px` 圆角
- 次要按钮: `#EDF2F8` 背景，`#4F6FAE` 文字
- 危险按钮: 透明背景，`#E74C3C` 文字，需二次确认

### 6.3 输入框
- 快速添加默认使用白色细边框，聚焦增加浅蓝 focus ring
- 描述输入使用 `#F3F5F7` 背景和 `10px` 圆角
- 详情基础字段默认透明，仅在 hover 或 focus 时显示浅灰背景

### 6.4 标签 Pill
- 背景 `#F0F0F4`，文字 `#666`，`12px` 全圆角
- 可附带删除 × 按钮

### 6.5 顶部工具
- 快速添加保持常驻。
- 搜索由图标按钮展开；有搜索内容时保持展开并提供清除按钮。
- 排序显示当前方式，通过下拉菜单切换，避免常驻五个排序按钮。

### 6.6 保存状态指示
- 已修改: 橙色圆点
- 保存中: 旋转动画
- 已保存: 绿色对勾，2 秒后消失
- 保存失败: 红色感叹号 + 重试按钮

## 7. 动画

- 持续时间: 120ms–180ms
- 缓动: `ease-out`
- 应用场景: hover 颜色过渡、面板展开/收起、详情切换、checkbox 状态切换
- 不使用: 弹跳、大幅度位移、旋转入场

## 8. 图标

- 使用 Lucide React 图标库
- 尺寸: 16px（行内）/ 20px（导航）
- 颜色: 继承当前文字色或 `#999`

## 9. 空状态

各视图空状态使用统一格式:
- 图标（48px，`#DDD`）
- 标题（13px，`#999`）
- 描述（11px，`#BBB`）

## 10. 错误状态

- API 不可用: 全屏连接错误页，居中显示图标 + "无法连接到服务" + 重试按钮
- 保存失败: 输入框边框变红，底部显示错误信息 + 重试链接
- 表单校验: 字段下方红色提示文字

## 11. 无障碍

- 所有图标按钮提供中文 `aria-label`
- 焦点样式: `2px solid #4F6FAE` outline，`2px` offset
- 键盘导航: Tab 键顺序与视觉顺序一致
- 颜色对比度: 所有文字与背景对比度 ≥ 4.5:1
