# Todo List 视觉设计方案

**日期:** 2026-06-11
**状态:** 已确认

## 1. 设计方向

**极简清爽** — 接近 Notion / Linear 风格，大量留白，灰白层次，单一主题色，温和圆角，阴影分层。

## 2. 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主题色 | `#6C5CE7` | 选中态、按钮、链接、checkbox |
| 主题浅色 | `#F4F2FD` | 选中行背景、hover 态 |
| 主题深色 | `#5B4FD9` | 按钮 hover、按压态 |
| 页面背景 | `#F8F9FB` | 全局底色 |
| 卡片/面板背景 | `#FFFFFF` | 中栏、右栏 |
| 侧栏背景 | `#F8F8FC` | 左栏微紫灰底 |
| 边框色 | `#EFEFF4` | 列分隔、卡片边框 |
| 主文字 | `#222222` | 标题 |
| 次文字 | `#555555` | 菜单项、标签 |
| 辅助文字 | `#999999` | 计数、占位符 |
| 禁用/占位 | `#CCCCCC` | 搜索占位、禁用态 |
| 高优先级 | `#E67E3A` | 高优先级标记 |
| 中优先级 | `#F0C45A` | 中优先级标记 |
| 完成/删除 | `#999999` + 删除线 | 已完成任务 |
| 标签背景 | `#F0F0F4` | 标签 pill |

## 3. 字体

- 主字体: `Segoe UI, system-ui, -apple-system, sans-serif`
- 代码/等宽: `Cascadia Code, Consolas, monospace`
- 字号层级: 10/11/12/13/15px（不设 14px 以保持视觉节奏）
- 行高: 1.5（正文），1.3（紧凑列表）

## 4. 圆角与阴影

| 元素 | 圆角 |
|------|------|
| 卡片/面板 | 10px |
| 按钮/输入框 | 7px |
| 选中行背景 | 6px |
| 标签 pill | 12px（全圆角） |
| checkbox | 4px |
| 小徽章 | 3px |

阴影: `0 2px 12px rgba(0,0,0,0.06)` — 仅卡片使用，轻微分层。

## 5. 布局参数

- 左栏展开宽度: `240px`，收起宽度: `56px`
- 中栏最小宽度: `420px`
- 右栏默认宽度: `420px`，无选中任务时隐藏
- 全局内边距: `16px`
- 列表项间距: `2px`
- 区块间距: `12-16px`

## 6. 组件样式

### 6.1 Checkbox
- 未选中: `16×16px`，`2px solid #D0D0DA` 边框，`4px` 圆角
- 已选中: 填充 `#6C5CE7`，白色对勾
- hover: 边框变为 `#6C5CE7`

### 6.2 按钮
- 主按钮: 填充 `#6C5CE7`，白色文字，`7px` 圆角
- 次要按钮: 透明背景，`#6C5CE7` 文字
- 危险按钮: 透明背景，`#E74C3C` 文字，需二次确认

### 6.3 输入框
- 默认: `1px solid transparent`，背景 `#F4F4F8`，`7px` 圆角
- 聚焦: 边框 `#6C5CE7`，背景 `#FFFFFF`
- 占位符: `#CCCCCC`

### 6.4 标签 Pill
- 背景 `#F0F0F4`，文字 `#666`，`12px` 全圆角
- 可附带删除 × 按钮

### 6.5 优先级标识
- 无 `0`: 不显示
- 低 `1`: 蓝色点 `#4F8EF7`
- 中 `3`: 黄色点 `#F0C45A`
- 高 `5`: 橙色点 `#E67E3A`

### 6.6 保存状态指示
- 已修改: 橙色圆点
- 保存中: 旋转动画
- 已保存: 绿色对勾，2 秒后消失
- 保存失败: 红色感叹号 + 重试按钮

## 7. 动画

- 持续时间: 120ms–180ms
- 缓动: `ease-out`
- 应用场景: hover 颜色过渡、面板展开/收起、详情切换、checkbox 状态切换
- 不使用: 弹跳、大幅度位移、旋转入场

## 8. 图标

- 使用 Lucide React 图标库
- 尺寸: 16px（行内）/ 20px（导航）
- 颜色: 继承当前文字色或 `#999`

## 9. 空状态

各视图空状态使用统一格式:
- 图标（48px，`#DDD`）
- 标题（13px，`#999`）
- 描述（11px，`#BBB`）

## 10. 错误状态

- API 不可用: 全屏连接错误页，居中显示图标 + "无法连接到服务" + 重试按钮
- 保存失败: 输入框边框变红，底部显示错误信息 + 重试链接
- 表单校验: 字段下方红色提示文字

## 11. 无障碍

- 所有图标按钮提供中文 `aria-label`
- 焦点样式: `2px solid #6C5CE7` outline，`2px` offset
- 键盘导航: Tab 键顺序与视觉顺序一致
- 颜色对比度: 所有文字与背景对比度 ≥ 4.5:1
